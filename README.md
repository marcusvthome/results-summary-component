## Meu processo

### Tecnologias utilizadas

- HTML5 semântico
- CSS puro
- Flexbox
- CSS Grid
- Google Fonts (Hanken Grotesk)

### O que aprendi

Aprendi a estilizar componentes usando CSS puro, sem depender de frameworks como Bootstrap. Os maiores aprendizados foram entender como `fr` funciona no Grid, a diferença entre `align-self` e `margin-left: auto` no Flexbox, e como criar um círculo com gradiente usando apenas `border-radius: 50%`.

### Próximos passos

Quero continuar praticando estilização com CSS puro para ficar cada vez mais confiante, e começar a trabalhar com JavaScript para adicionar interatividade aos projetos.
